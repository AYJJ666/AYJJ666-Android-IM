### 即时通讯APP
使用环信IM提供的后端服务。实现联系人管理，单聊等功能。使用Kotlin语言编写，页面主要使用安卓原生控件实现效果。

#### 后端

使用[环信SDK](https://docs-im.easemob.com/ccim/android/quickstart)

### 注册/登录

自定义账号密码，由于环信SDK仅支持登录后更改当前账号属性，所以首次登录时设置随机昵称，使用默认头像。
